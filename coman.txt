# Команди для управління проектом Chicken Road

## Бекенд (Strapi)

Щоб запустити сервер бекенду, відкрийте термінал, перейдіть до папки `backend` і виконайте команду:

```bash
cd backend
npm run develop
```

Сервер буде доступний за адресою: http://localhost:1337
Адмін-панель: http://localhost:1337/admin

## Фронтенд (Next.js)

Щоб запустити сервер фронтенду, відкрийте інший термінал, перейдіть до папки `frontend` і виконайте команду:

```bash
cd frontend
npm run dev
```

Сайт буде доступний за адресою: http://localhost:3000
